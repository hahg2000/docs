# Vim

# Vim 三种基本模式

  1. 一般模式/命令模式：输入的任何东西都会解析成指令

  2. 插入模式/编辑模式 ：进入按“ i ”，退出进入一般模式按Esc

  3. 末行模式/底线命令模式：进入按 “ ：”，这时你的光标会移到屏幕的底部。在这里可以输入很多特定的指令。例如，保存文件、将光标跳转到特定的位置还有打开或者关闭Vim的选项和配置

# 快速开始 Vim

  1. 安装Vim
  2. 打开Command（cmd）窗口
  3. 输入 “ Vim ” ，就会进入Vim
  4. 退出Vim：在输入法为英文的状态下，按下“ : ”——进入末行模式，然后输入“ q! ”，再按下回车

# 创建并保存文件

  1. 打开Command（cmd）窗口
  2. 输入以下指令
vim abc1
1
  1. 按下 “ i ”，进入插入模式
  2. 输入你需要记录的信息
  3. 按下“ Esc ”，进入一般模式
  4. 按下“ : "，输入“ wq ”，w代表write，把你的修改写入文件；q代表quit，关闭Vim
